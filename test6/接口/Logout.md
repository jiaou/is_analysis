# 接口：Logout  [返回](../Readme.md)
用例： [登出](../用例/登出.md)

- 功能：
    已经登录的用户退出。

- 权限：
    用户已经登录。

- API请求地址：
    接口基本地址/v1/api/Logout

- 请求方式 ：
    POST

- 请求实例：

        {
            "user_id":0001
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |user_id|已经登录用户的user_id值，对应表USERS.USER_ID的值|

- 返回实例：

        {
            "status": true,
            "info": null,

        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
